# Job photos

Drop your job photos in this folder, then tell Claude (or edit
`src/components/work-gallery.tsx`) to display them.

Tips for good, fast-loading photos:
- **Format:** JPG or WebP
- **Size:** resize to about 1600px on the long edge and keep each file under
  ~1 MB. Phone photos are often 3–5 MB — too big for the repo and slow on mobile.
- **Names:** keep them clear, e.g. `bathroom-1.jpg`, `garden-patio.jpg`,
  `leak-repair.jpg`. Before/after pairs work great.

## ⚠️ Videos do NOT go here
GitHub rejects files over 100 MB and large media bloats the repo. Upload videos
to **YouTube** (set them to *Unlisted* if you don't want them searchable) or
**Vimeo**, then share the links — they'll be embedded on the site.
